Hex is a code that identifes the color, RGB uses different amounts of Red, Green and Blue to make different colors and HSL uses Hue, Saturation and Lightness to make different colors and shades of color.

Preferred color code format: I would prefer to use RBG becasue with RGB you can make any color you want.

HWB summary: HWB uses Hue, Whiteness and Blackness to make different colors and different shades of them. This is a new color standard in CSS4.

CMYK summary: CMYK uses cyan, magenta, yellow and black like what ink printers use to make any color.
